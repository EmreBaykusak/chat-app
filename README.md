# Chat Application

A real-time chat application built with Node.js and Socket.IO.

## Features
- Real-time messaging with Socket.IO
- Multiple chat rooms
- Location sharing via browser geolocation
- Profanity filter using bad-words
- Auto-scroll UX for new messages
- Vanilla JavaScript frontend with Node/Express backend

## Tech Stack
- Node.js
- TypeScript
- Express
- Socket.IO
- Vanilla JavaScript
- HTML / CSS
